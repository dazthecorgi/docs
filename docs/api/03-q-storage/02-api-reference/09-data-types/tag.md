# Tag

Container for a tag key and value.

## Contents

### Key

Name of the tag.

**Type**: String  
**Required**: Yes

### Value

Value of the tag.

**Type**: String  
**Required**: Yes

## Example

```xml
<Tag>
   <Key>Project</Key>
   <Value>Project One</Value>
</Tag>
```
